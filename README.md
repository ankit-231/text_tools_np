# text_tools_np

## Unicode to Preeti converter

### My Plan

- Open and read word file
- Get the text
- Send the text via an online text converter
- Parse the response
- Write to a new file

### Tech Stack

- python-docx
- requests
- beautifulsoup4
- lxml

### Getting Started

#### Create a virtual environment

```bash
python3 -m venv .venv
source venv/bin/activate
```

#### Install the dependencies

```bash
pip install -r requirements.txt
```

#### Run the script

```bash
python main.py
```
